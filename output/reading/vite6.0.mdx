## Vite 6.０が来るらしい

https://vite.dev/blog/announcing-vite6

> It has been an eventful year. Vite adoption keeps growing, with npm downloads per week jumping from 7.5 million to 17 million since the release of Vite 5 a year ago

> 今年は波乱に富んだ年でした。Vite の採用は増え続けており、1 年前の Vite 5 のリリース以来、npm のダウンロード数は週あたり 750 万から 1,700 万に急増しています。

すごい！（小並感）

大学生活が終わって今の会社に入社した頃に話題になり始めてるな〜って記憶で止まってしまってた。もう一般的なビルドツールになってるのね。。。ちゃんとアンテナ張らないと。。

一応使っていたこともあったけど、まだまだWebpackを使っているような所も多くて保守管理とかはViteあんまり見かけなかった記憶。

> Vite 6 is the most significant major release since Vite 2. We're eager to partner with the ecosystem to keep expanding our shared commons through new APIs and, as usual, a more polished base upon which to build.

> Vite 6 は、Vite 2 以来最も重要なメジャーリリースです。私たちは、エコシステムと提携して、新しいAPIと、いつものように、より洗練された基盤を通じて共有地を拡大し続けたいと考えています。

### ライブエディタ

```jsx live
function Clock(props) {
  const [date, setDate] = useState(new Date())
  useEffect(() => {
    const timerID = setInterval(() => tick(), 1000)

    return function cleanup() {
      clearInterval(timerID)
    }
  })

  function tick() {
    setDate(new Date())
  }

  return (
    <div>
      <h2>It is {date.toLocaleTimeString()}.</h2>
    </div>
  )
}
```
